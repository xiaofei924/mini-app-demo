
## 概述

为了帮助开发者快速上手小程序的开发，熟悉小程序IDE的常用功能，我们提供了一个简单的Demo小程序。

现在开始去完成右侧的模拟器任务，体验真正的小程序开发吧！

## 流程

请按照右侧的步骤提示来获取小程序开发体验证书。
